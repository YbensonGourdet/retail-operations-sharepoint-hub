# Document Lifecycle

| Stage | Status | Description |
|---|---|---|
| Draft | Draft | Document is being created or updated |
| Review | Under Review | Process owner is checking accuracy |
| Published | Active | Document is approved and available for use |
| Maintenance | Active | Document remains available but is monitored for changes |
| Retired | Archived | Document is no longer used as the main source |

## Review Cycle

1. Process owner reviews active documents every six months.
2. Review Date is updated in the SOP Library metadata.
3. If the process is still valid, the document remains Active.
4. If updates are needed, the document is marked Under Review.
5. Once updates are approved, the document is republished as Active.
6. Outdated versions are moved to Archived.
