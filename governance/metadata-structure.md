# Metadata Structure

The SOP Library uses metadata to make documents easier to search, filter, manage, and review.

| Metadata Field | Purpose |
|---|---|
| Process Area | Groups documents by operational workflow |
| Document Type | Identifies whether content is an SOP, checklist, guide, FAQ, or process map |
| Status | Shows whether content is Draft, Active, Under Review, or Archived |
| Audience | Helps users find documents relevant to their role |
| Difficulty | Supports onboarding and training navigation |
| Review Date | Tracks when the document should be reviewed |
| Process Owner | Identifies who maintains the document |
| Business Impact | Helps prioritize important processes and updates |
